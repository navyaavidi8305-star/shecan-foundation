# She Can Foundation – Contact Form Application

## Project Overview

This is a full-stack contact form application developed using the MERN stack. Users can submit their name, email, and message through a responsive form interface.

---

## Features

- Responsive Contact Form
- MongoDB Database Integration
- REST API Integration
- Success Message After Submission
- Modern UI Design

---

## Technologies Used

### Frontend
- React.js
- Tailwind CSS
- Axios
- Vite

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
